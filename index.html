How to use this (read once) ›
  • 1. Find a tweet on @BraRicky's page (or a big account we're replying to) that needs a reply.
  • 2. Copy that tweet and paste it in the box below.
  • 3. Pick which reply moods you want. Leave all on if unsure.
  • 4. Tap Build the prompt, then Copy.
  • 5. Tap ChatGPT / Claude / Gemini, paste, and send. You'll get several ready-to-post replies in Richard's voice.
  • 6. Pick the best one, tweak if needed, post it.

Works on iPhone and Android. To keep it handy: open in your browser, then "Add to Home Screen" so it behaves like an app.

Reply do's & don'ts (the rules) ›

The voice

Calm, grounded, elder-brother wisdom. The disciplined man who's building something real, never the loud "alpha." Short, declarative, complete thoughts. Cut it in half, then post.

Do

  • Be specific to the actual tweet - add real value.
  • Keep it short and quotable. One clean idea.
  • Speak to financial pressure with respect, not hype.
  • Use the calm, slightly spiritual "make-it-out" frame.

Don't

  • No generic copy-paste lines like "Goodluck to everyone."
  • No fake Nigerian slang Richard doesn't actually use.
  • No hashtags, no spam emojis, no "great post!"
  • No loud alpha bragging or bitterness.
